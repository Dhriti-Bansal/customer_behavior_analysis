# Data Analytics Project

## Overview
This project demonstrates an end-to-end data analytics workflow, starting from raw data and finishing with business insights and presentation-ready outputs. The project includes data loading in Python, exploratory data analysis (EDA), data cleaning, SQL analysis using a relational database, dashboard development in Power BI, and final reporting through a written report and presentation.

The goal of this project is to show the full analytics process in a practical and structured way, using industry-relevant tools and techniques.

---

## Dataset
The dataset used in this project contains structured data suitable for analysis, reporting, and dashboard creation. It was imported into Python for initial exploration and cleaning, then loaded into a SQL database for querying and deeper analysis.

### Dataset tasks completed:
- Loaded dataset into Python
- Checked data types and structure
- Identified missing values and duplicates
- Cleaned and prepared data for analysis
- Exported cleaned data for SQL and dashboard use

## Project Steps

### 1. Data Loading in Python
The dataset was imported into Python and reviewed to understand its structure, column names, data types, and overall quality.

### 2. Exploratory Data Analysis (EDA)
EDA was performed to better understand the dataset and identify trends, patterns, and possible issues. This included:
- Summary statistics
- Distribution checks
- Missing value analysis
- Duplicate detection
- Basic visualizations

### 3. Data Cleaning
The dataset was cleaned to improve accuracy and usability. Cleaning steps included:
- Handling missing values
- Removing duplicates
- Fixing inconsistent formatting
- Converting data types where needed
- Preparing the dataset for SQL analysis and dashboarding

### 4. SQL Analysis
After cleaning, the data was loaded into a SQL database. SQL queries were written to answer business questions and generate insights. This stage included:
- Creating tables
- Importing cleaned data
- Writing SELECT, WHERE, GROUP BY, ORDER BY, JOIN, and aggregate queries
- Extracting key metrics and trends

### 5. Dashboard Development
A Power BI dashboard was created to present the key findings in a visual and interactive format. The dashboard was designed to make insights easy to understand for both technical and non-technical audiences.

### 6. Reporting and Presentation
The final stage of the project involved summarizing the analysis in a written report. 
This helped communicate the workflow, findings, and recommendations in a professional format.
